About Our Simple Content Management System (CMS)

We have built a simple Content Management System (CMS) using Angular, a modern front-end framework. The CMS includes dynamic components such as menus, sidebars, modals, and customizable page content, all seamlessly managed and controlled. Key technologies used in this system include:

  • Angular: A component-based framework to build scalable, dynamic user interfaces.
  • TypeScript: For structured and strongly typed JavaScript code.
  • Bootstrap: For responsive and visually appealing UI design.
  • RxJS (Reactive Extensions for JavaScript): To handle reactive state management and observables.
  • Custom Modal Components and Services: To enable global control of modals across the application.

This architecture allows components like <app-header-menu>, <app-side-menu>, and modals to be reused and dynamically updated, making the application flexible and maintainable.
