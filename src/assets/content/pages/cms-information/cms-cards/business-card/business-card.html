BusinessCardComponent Instructions

The BusinessCardComponent displays professional business cards for individuals by fetching their details from a JSON file. Each card includes a profile photo, name, contact information (business phone, cell, email), social media links, and a brief biography.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON file containing business card data (e.g., 'business-card-data.json'). The file is loaded from the assets folder.
divId String An identifier for the main container of the business cards, used for styling or DOM targeting.

JSON Data Structure

The component expects the JSON file to have the following structure: 


{
  "people": [
      {
        "name": "John Doe",
        "photo": "./assets/images/motorcycle.png",
        "businessPhone": "123-456-7890",
        "cellPhone": "987-654-3210",
        "email": "john.doe@example.com",
        "socialMedia": [
          { "icon": "bi-facebook", "url": "https://facebook.com/johndoe" },
          { "icon": "bi-twitter", "url": "https://twitter.com/johndoe" }
        ],
        "bio": "John is a software developer with 10 years of experience."
      },
      {
        "name": "Jane Smith",
        "photo": "./assets/images/motorcycle.png",
        "businessPhone": "234-567-8901",
        "cellPhone": "876-543-2109",
        "email": "jane.smith@example.com",
        "socialMedia": [
          { "icon": "bi-linkedin", "url": "https://linkedin.com/in/janesmith" },
          { "icon": "bi-instagram", "url": "https://instagram.com/janesmith" }
        ],
        "bio": "Jane is a graphic designer passionate about branding and UI/UX." 
      },
      {
        "name": "Michael Brown",
        "photo": "./assets/images/motorcycle.png",
        "businessPhone": "345-678-9012",
        "cellPhone": "765-432-1098",
        "email": "michael.brown@example.com",
        "socialMedia": [
          { "icon": "bi-github", "url": "https://github.com/michaelbrown" },
          { "icon": "bi-twitter", "url": "https://twitter.com/michaelbrown" }
        ],
        "bio": "Michael is a data scientist specializing in AI and machine learning."
      }
  ]
}

In this JSON structure, each person object includes: name – the individual's full name, photo – the path to the profile image, businessPhone – the business contact number, cellPhone – the mobile phone number, email – the email address, socialMedia – an array of social media objects (each with icon and url), bio – a brief biography.
