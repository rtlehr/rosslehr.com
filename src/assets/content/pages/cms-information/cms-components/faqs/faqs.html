FaqComponent Instructions

The FaqComponent is designed to display a list of frequently asked questions along with their answers. It renders the questions in a clickable list and, when a question is selected, smoothly scrolls to the corresponding answer. Each answer section also includes a "Back to Top" link for easy navigation.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON file containing FAQ data (e.g., 'faq-data.json'). The file is loaded from the assets folder.
divId String An identifier for the main container of the FAQ component, which can be used for styling or DOM manipulation.

JSON Data Structure

The component expects the JSON file to be an array of FAQ objects with the following structure: 


[
    {
        "id": "1",
        "question": "What is your return policy?",
        "answer": "You can return any item within 30 days of purchase with a receipt."
    },
    {
        "id": "2",
        "question": "How can I track my order?",
        "answer": "You can track your order by logging into your account and visiting the 'Orders' section."
    },
    {
        "id": "3",
        "question": "Do you offer gift cards?",
        "answer": "Yes, we offer digital and physical gift cards available for purchase online."
    },
    {
        "id": "4",
        "question": "What payment methods do you accept?",
        "answer": "We accept major credit cards, PayPal, and Apple Pay."
    },
    {
        "id": "5",
        "question": "Can I cancel or modify my order?",
        "answer": "Yes, you can cancel or modify your order within 24 hours of placing it by contacting customer support."
    }
]

Each FAQ object includes: id – a unique identifier, question – the FAQ question, answer – the corresponding answer.
