InfoHighlightComponent Instructions

The InfoHighlightComponent is an interactive image slider designed to highlight key information through clickable text boxes. It loads configuration data from a JSON file, which defines the overall component size, auto-scroll behavior, and an array of images. Each image object includes an image URL, a title, and an optional text overlay. Users can manually change the displayed image by clicking on the corresponding text box, and if enabled, the slider will auto-scroll through the images.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON configuration file that provides the slider settings and image data (e.g., 'info-highlight-data.json'). The file is loaded from the assets folder.
divId String An identifier for the main container of the slider, useful for custom styling or DOM targeting.

JSON Data Structure

The component expects the JSON file to have the following structure: 


{
  "componentSize": 500,
  "autoScroll": true,
  "images": [
      {
        "imageUrl": "./assets/images/motorcycle.png",
        "title": "This is Box 1",
        "textBoxes": []
      },
      {
        "imageUrl": "./assets/images/snowtruck.jpg",
        "title": "This is Box 2",
        "textBoxes": [
          {
            "text": "fringilla mi vel, fermentum massa. Integer mollis nunc erat, nec aliquet quam mattis vestibulum."
          }
        ]
      },
      {
        "imageUrl": "./assets/images/motorcycle.png",
        "title": "This is Box 3",
        "textBoxes": [
          {
            "text": "Aenean ex libero, ullamcorper non neque in, vulputate dignissim nisi. Nam tincidunt sagittis lacus, sit amet semper odio aliquet eu."
          }
        ]
      },
      {
        "imageUrl": "./assets/images/snowtruck.jpg",
        "title": "This is Box 4",
        "textBoxes": [
          {
            "text": "bibendum urna. Suspendisse tristique metus quis viverra commodo."
          }
        ]
      },
      {
        "imageUrl": "./assets/images/motorcycle.png",
        "title": "This is Box 5",
        "textBoxes": []
      }
  ]
}

In this structure, componentSize determines the dimensions of the slider, autoScroll enables or disables automatic image transitions, and images is an array where each object specifies the image URL, a title to display on the text box, and an optional textBoxes array for overlay text.
