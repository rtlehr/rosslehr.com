How to Use and Activate the Modal Window

The modal window can be triggered manually using a button or link, or automatically when loading content.

Understanding the data-content JSON Structure

The data-content attribute contains a JSON object that defines the modalâ€™s properties. Below is an explanation of each key:

Property Type Description
title String The title displayed in the modal window.
divId String The ID of the <div> where dynamic content will be loaded (if applicable).
contentFile String A file path pointing to an external HTML file to be loaded inside the modal.

Example 1: Button to Open Modal

The following button opens a modal window with predefined settings:

      
<button data-click="openModalWindow" 
      data-content='{  
        "title": "Hello from Button",
        "contentFile": "content/pages/cms-information/modal-window/button-modal.html",
        "divId": "modalWindowContent",
      }'>
  Open Modal
</button>

Example 2: Link to Open Modal

You can also open the modal using a standard hyperlink:

      
<a href="#" data-click="openModalWindow" 
 data-content='{  
   "title": "Hello from Link",
   "contentFile": "content/pages/cms-information/modal-window/anchor-modal.html",
   "divId": "modalWindowContent"
 }'>
  Open Modal from Link
</a>
Open Modal from Link

Automatically Triggering the Modal on Page Load

If you want the modal to open automatically when a content file is loaded, include a <trigger-modal> tag inside the HTML file. 

      
<trigger-modal data-content='{   "title": "Auto Triggered Modal Window",
                                  "divId": "modalWindowContent", 
                                  "contentFile": "content/pages/cms-information/modal-window/auto-triggered-modal.html"
                                  }'>
                                  </trigger-modal>

When the content containing <trigger-modal> is loaded, the modal will automatically display with the provided message.
