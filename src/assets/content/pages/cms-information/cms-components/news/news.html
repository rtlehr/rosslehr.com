NewsComponent Instructions

The NewsComponent is designed to fetch and display a list of news articles using Angular's HttpClient. It loads the news data from a JSON file specified by the fileToLoad input and renders each news item with a thumbnail image, title, publication date, an excerpt of the content, and a "Read More" button.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON file containing the news data (e.g., 'news-data.json'). The file is fetched from the assets folder.
divId String An identifier for the main news list container, which can be used for custom styling or DOM manipulation.

JSON Data Structure

The component expects the JSON file to be an array of news objects with the following structure: 


[
    {
      "id": "1",
      "title": "Breaking News: Angular Rocks!",
      "image": "https://via.placeholder.com/150",
      "date": "December 11, 2024",
      "content": "Angular has released its latest version with amazing new features and improvements. Angular has released its latest version with amazing new features and improvements..."
    },
    {
      "id": "2",
      "title": "Tech Trends for 2024",
      "image": "https://via.placeholder.com/150",
      "date": "December 5, 2024",
      "content": "Explore the top tech trends of the year, including AI, blockchain, and more."
    },
    {
      "id": "3",
      "title": "Community Spotlight: Open Source Contributions",
      "image": "https://via.placeholder.com/150",
      "date": "November 30, 2024",
      "content": "Highlighting the most significant contributions from the open-source community this year."
    }
]

Each news object includes: id – a unique identifier, title – the headline, image – the URL for the thumbnail image, date – the publication date, and content – an excerpt of the article.
