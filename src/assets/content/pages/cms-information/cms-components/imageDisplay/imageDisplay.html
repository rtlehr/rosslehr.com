ImageDisplayComponent Instructions

The ImageDisplayComponent is designed to display a gallery of images loaded from a JSON data file. It renders clickable thumbnails for public images and a placeholder for private images. When a public image is clicked, the component opens a modal window displaying a larger version of the image along with its title and description.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON file containing image data (e.g., 'images-data.json'). The file is loaded from the assets folder.
divId String An identifier assigned to the main gallery container for targeting or styling purposes.

JSON Data Structure

The component expects the JSON file to be an array of image objects with the following structure: 


[
    {
        "id": "1",
        "src": "./assets/images/flag.jpeg",
        "largeSrc": "./assets/images/flag.jpeg",
        "title": "Image 1",
        "description": "This is the description for Image 1.",
        "private": "false" 
    },
    {
        "id": "2",
        "src": "./assets/images/airplane.jpeg",
        "largeSrc": "./assets/images/airplane.jpeg",
        "title": "Image 1",
        "description": "This is the description for Image 1.",
        "private": "true"
    },
    {
        "id": "3",
        "src": "./assets/images/motorcycle.png",
        "largeSrc": "./assets/images/motorcycle.png",
        "title": "Image 1",
        "description": "This is the description for Image 1.",
        "private": "false"
    }
]

In this structure, the src property holds the path for the thumbnail image, while the largeSrc is used in the modal view. The private property determines whether the actual image or a placeholder image should be displayed.
