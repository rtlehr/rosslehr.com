ImageSliderComponent Instructions

The ImageSliderComponent displays a responsive image slider that automatically cycles through images. The slider's settings and images are loaded from a JSON configuration file. It features navigation buttons for manual control and smooth transitions between slides.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON configuration file containing slider settings and images (e.g., 'slider-data.json').
sliderWidth String The CSS width for the slider (e.g., '800px' or '100%'). This value can be overridden by the JSON configuration.
sliderHeight String The CSS height for the slider (e.g., '400px'). This value can be overridden by the JSON configuration.
imagesCount Number The total number of images loaded into the slider. This value is set based on the JSON file.
divId String An identifier for the main slider container, useful for styling or DOM manipulation.

JSON Data Structure

The component expects the JSON configuration file to have the following structure: 


{
  "width": "800px",
  "height": "400px",
  "slideInterval": 3000,
  "images": [
    {
      "src": "./assets/images/flag.jpeg",
      "alt": "Image 1"
    },
    {
      "src": "./assets/images/airplane.jpeg",
      "alt": "Image 2"
    },
    {
      "src": "./assets/images/motorcycle.png",
      "alt": "Image 3"
    }
  ]
}
