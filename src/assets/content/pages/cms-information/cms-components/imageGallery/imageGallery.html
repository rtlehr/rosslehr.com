ImageGalleryComponent Instructions

The ImageGalleryComponent displays a gallery of images in a responsive, card-based layout using Bootstrap classes. Each image card contains an image, title, short description, and a "Learn More" button linking to an external resource.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON file containing image data (e.g., 'image-data.json'). The file is loaded from the assets folder.
divId String An identifier assigned to the main container of the gallery, useful for styling or DOM targeting.

JSON Data Structure

The component expects the JSON file to be an array of image objects with the following structure: 


[
    {
      "src": "./assets/images/motorcycle.png",
      "title": "Image 1",
      "description": "This is a short description for Image 1.",
      "href": "https://example.com/image1"
    },
    {
      "src": "./assets/images/snowtruck.jpg",
      "title": "Image 1",
      "description": "This is a short description for Image 1.",
      "href": "https://example.com/image1"
    },
    {
      "src": "./assets/images/flag.jpeg",
      "title": "Image 1",
      "description": "This is a short description for Image 1.",
      "href": "https://example.com/image1"
    }
]

In this structure, the src property specifies the path of the image file, title and description provide the textual content for the card, and href defines the URL for the "Learn More" link.
