InfoCardsComponent Instructions

The InfoCardsComponent displays a set of information cards arranged in a grid layout. Each card features an icon, a title, and a description, and is wrapped in an anchor tag that can link to additional information. The component dynamically loads the card data from a JSON file using a JSON data service.

Component Inputs

Input Name Type Description
fileToLoad String The path to the JSON file containing the information card data (e.g., 'content/pages/cms-information/component-samples/info-cards/info-cards.json').
divId String The identifier for the main container of the info cards, used for styling or DOM targeting.

JSON Data Structure
